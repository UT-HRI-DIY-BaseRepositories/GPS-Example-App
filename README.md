# GPS_Example_App
This is one of the base repositories for Lessons 5.1 - 5.5. In other words, this is one of the project pathways for anyone that wants to do a GPS based app.
<br>
<br>
Be sure to change your device settings to enable GPS tracking for this app to work. For example, if you are using an iPhone and are using Safari as your browser, then you should first go to your Settings app --> Location Services --> Safari Websites --> "Ask Next Time or When I Share". 
<br>
<br>
If you have tried the above step and the GPS tracking does not work, you can try to use other browsers (such as Chrome or Microsoft Edge).
